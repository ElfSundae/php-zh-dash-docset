APCIterator::next

(PECL apc >= 3.1.1)

APCIterator::next — 移到下一项

说明

public void APCIterator::next ( void )

移动迭代器的指针到下一个元素。

参数

此函数没有参数。

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。

参见

  • APCIterator::current() - 获取当前项
  • APCIterator::rewind() - 倒退迭代器
  • Iterator::next() - 向前移动到下一个元素
