counter_get_meta

counter_get_meta — 返回计数器资源的部分元信息。

说明

mixed counter_get_meta ( resource $counter , integer $attribute )

counter_get_meta() 返回计数器资源的元信息。

参数

counter
要操作的计数器资源。
attribute
要取回的元信息。

返回值

counter_get_meta() 根据所请求的元数据返回不同类型的值。
