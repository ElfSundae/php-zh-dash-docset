Exception::__clone

(PHP 5 >= 5.1.0, PHP 7)

Exception::__clone — 异常克隆

说明

final private void Exception::__clone ( void )

尝试克隆异常，这将导致一个致命错误。

参数

此函数没有参数。

返回值

没有返回值。

错误／异常

异常被不允许克隆。
