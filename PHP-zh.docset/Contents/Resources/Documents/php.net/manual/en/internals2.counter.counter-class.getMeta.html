Counter::getMeta

Counter::getMeta — 返回计数器的部分元信息。

说明

mixed Counter::getMeta ( integer $attribute )

Counter::getMeta() 返回一个计数器的元信息。

参数

attribute
要取回的元信息。

返回值

Counter::getMeta() 根据所请求的元数据返回不同类型的值。
