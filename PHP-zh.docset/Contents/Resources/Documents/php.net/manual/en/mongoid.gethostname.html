MongoId::getHostname

(PECL mongo >= 1.0.8)

MongoId::getHostname — 获取这台机器上 id 所使用的 hostname

说明

public static string MongoId::getHostname ( void )

返回的 hostname 用于 MongoId 产生唯一 id。 这和 gethostname() 应该会返回同样的值。

它和以下函数是一样的：

<?php

public static function getHostname() {
    return gethostname();
}

?>

参数

此函数没有参数。

返回值

返回 hostname。
