counter_get_named

counter_get_named — 按名称查询一个已存在的计数器，并作为资源返回。

说明

resource Counter::getNamed ( string $name )

counter_get_named() 返回一个存在的计数器名称名称不存在则返回NULL。

参数

name
要查询的计数器名称。

返回值

counter_get_name() 返回一个存在的计数器名称，名称不存在则返回NULL。
