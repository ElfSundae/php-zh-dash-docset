strrev

(PHP 4, PHP 5, PHP 7)

strrev — 反转字符串

说明

string strrev ( string $string )

返回 string 反转后的字符串。

参数

string

待反转的原始字符串。

返回值

返回反转后的字符串。

范例

Example #1 使用 strrev() 反转字符串

<?php
echo strrev("Hello world!"); // 输出 "!dlrow olleH"
?>
