Threaded::count

(PECL pthreads >= 2.0.0)

Threaded::count — Manipulation

说明

public integer Threaded::count ( void )

返回对象的属性数量

参数

此函数没有参数。

返回值

范例

Example #1 计算对象中的属性数量

<?php
$safe = new Threaded();

while (count($safe) < 10) {
    $safe[] = count($safe);
}

var_dump(count($safe));
?>

以上例程会输出：

int(10)
