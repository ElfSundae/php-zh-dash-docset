bzflush

(PHP 4 >= 4.0.4, PHP 5)

bzflush — 强制写入所有写缓冲区的数据

说明

int bzflush ( resource $bz )

强制写入 bzip2 文件指针 bz 的所有写缓冲数据。

参数

bz

文件指针。它必须是有效的并且指向 bzopen() 成功打开的文件。

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。

参见

  • bzread() - bzip2 文件二进制安全地读取
  • bzwrite() - 二进制安全地写入 bzip2 文件
