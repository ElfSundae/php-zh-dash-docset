counter_bump_value

counter_bump_value — 更新计数器资源的当前值。

说明

void counter_bump_value ( resource $counter , integer $offset )

counter_bump_value() 更新计数器资源的当前值。

参数

counter
要操作的计数器资源。
offset
计数器值的变化量。可为负数。

参见

  • counter_get_value() - 获取计数器资源的当前值。
  • counter_reset_value() - 重置计数器资源的当前值。
