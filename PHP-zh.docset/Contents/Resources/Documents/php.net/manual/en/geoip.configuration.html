运行时配置

这些函数的行为受 php.ini 中的设置影响。

GeoIP 配置选项
名字 默认 可修改范围 更新日志
geoip.custom_directory "" PHP_INI_ALL

这是配置指令的简短说明。

geoip.custom_directory string

Empty by default, but can be set to force a different database path than the one compiled in the library.
