rad2deg

(PHP 4, PHP 5, PHP 7, PHP 8)

rad2deg — 将弧度数转换为相应的角度数

说明

rad2deg ( float $number ) : float

本函数将 number 从弧度转换为角度。

参数

number

一个弧度值

返回值

number 相应的角度数

范例

Example #1 rad2deg() 例子

<?php

echo rad2deg(M_PI_4); // 45

?>
