Throwable::getTrace

(PHP 7, PHP 8)

Throwable::getTrace — 获取堆栈踪迹（Stack Trace）

说明

public Throwable::getTrace ( ) : array

返回 array 的堆栈踪迹（Stack Trace）。

参数

此函数没有参数。

返回值

返回 array 的堆栈踪迹（Stack Trace），和 debug_backtrace() 具有相同的格式。
