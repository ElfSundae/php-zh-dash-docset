rrd_info

(PECL rrd >= 0.9.0)

rrd_info — 获取 rrd 文件有关信息

说明

rrd_info(string $filename): array

返回有关 RRD 数据库文件的特定信息。

参数

file

RRD 数据库文件名。

返回值

有关请求的 RRD 文件的信息数组， 或者在失败时返回 false。
