ReflectionFunctionAbstract::getNamespaceName

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

ReflectionFunctionAbstract::getNamespaceName — 获取命名空间

说明

public ReflectionFunctionAbstract::getNamespaceName(): string

获取类定义所属的命名空间。

参数

此函数没有参数。

返回值

返回命名空间。
