Error::__clone

(PHP 7, PHP 8)

Error::__clone — 克隆 error

说明

private Error::__clone(): void

Error 无法被克隆，调用该方法会导致 fatal error。

参数

此函数没有参数。

返回值

没有返回值。

错误／异常

Error 无法被克隆。

更新日志

版本 说明
8.1.0 Error::__clone() 不再是 final 的。
