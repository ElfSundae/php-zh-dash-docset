简介

mysqli 扩展允许访问 MySQL 4.1 及以上版本提供的功能。关于 MySQL 数据库服务器的更多信息请参阅 » http://www.mysql.com/。

可在 概述 中找到从 PHP 中使用 MySQL 的软件概述。

MySQL 的文档请查看 » http://dev.mysql.com/doc/。

本文档中的部分内容摘自 Oracle Corporation 许可的 MySQL 手册。

示例使用免费提供的 » world 或 » sakila 数据库。
