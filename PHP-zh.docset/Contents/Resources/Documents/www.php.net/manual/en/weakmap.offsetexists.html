WeakMap::offsetExists

(PHP 8)

WeakMap::offsetExists — 检测 map 中是否存在某个对象

说明

public WeakMap::offsetExists ( object $object ) : bool

检查传入的对象在 map 中是否被引用。

参数

object

要检查的对象。

返回值

如果 map 中包含该对象返回 true，否则返回 false。
