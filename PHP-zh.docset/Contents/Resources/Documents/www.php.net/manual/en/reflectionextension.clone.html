ReflectionExtension::__clone

(PHP 5, PHP 7, PHP 8)

ReflectionExtension::__clone — 克隆

说明

private ReflectionExtension::__clone(): void

clone 方法阻止对象克隆。不允许克隆反射对象。

参数

此函数没有参数。

返回值

没有返回值，如果被调用将产生 fatal 错误。

更新日志

版本 说明
8.1.0 此函数不再是 final。
