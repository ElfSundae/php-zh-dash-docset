DateTime::getLastErrors

date_get_last_errors

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

DateTime::getLastErrors -- date_get_last_errors — 返回警告和错误信息

说明

面向对象风格

public static DateTime::getLastErrors(): array|false

过程化风格

date_get_last_errors(): array|false

跟 DateTimeImmutable::getLastErrors() 一样，但适用于 DateTime。

参数

此函数没有参数。

返回值

返回包含警告和错误信息的数组，如果没有任何警告或者错误，则返回 false。

参见

  • DateTimeImmutable::getLastErrors() - Returns the warnings and errors
