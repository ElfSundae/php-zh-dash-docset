安装

从 PHP 4.4.0 开始，这些函数默认是被激活的. 对于旧版本在配置和编译 PHP 是使用 --enable-tokenizer选项. 也能使用 --disable-tokenizer选项禁用 tokenizer 支持.

PHP 的 Windows 版本已内建对此扩展的支持。不需要载入额外的扩展来使用这些函数。

Note: tokenizer 的内建支持在 PHP 4.3.0 后可用
