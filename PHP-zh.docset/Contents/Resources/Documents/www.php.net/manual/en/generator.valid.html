Generator::valid

(PHP 5 >= 5.5.0, PHP 7)

Generator::valid — 检查迭代器是否被关闭

说明

public Generator::valid ( void ) : bool

参数

此函数没有参数。

返回值

如果迭代器已被关闭返回 FALSE，否则返回 TRUE。
