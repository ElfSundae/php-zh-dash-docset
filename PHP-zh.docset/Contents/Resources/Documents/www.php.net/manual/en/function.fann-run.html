fann_run

(PECL fann >= 1.0.0)

fann_run — 将通过神经网络运行输入。

说明

fann_run(resource $ann, array $input): array

将通过神经网络运行输入，返回输出数组，数组的长度等于输出层神经元的个数。

参数

ann

神经网络 资源。

input

输入数组的值。

返回值

成功，返回输出数组，错误则返回 false .
