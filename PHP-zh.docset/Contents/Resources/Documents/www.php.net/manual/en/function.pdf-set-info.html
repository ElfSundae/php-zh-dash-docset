PDF_set_info

(PHP 4 >= 4.0.1, PECL pdflib >= 1.0.0)

PDF_set_info — Fill document info field

说明

PDF_set_info ( resource $p , string $key , string $value ) : bool

Fill document information field key with value. 成功时返回 TRUE， 或者在失败时返回 FALSE。
