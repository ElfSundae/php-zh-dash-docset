pg_untrace

(PHP 4 >= 4.0.1, PHP 5, PHP 7, PHP 8)

pg_untrace — 关闭 PostgreSQL 连接的追踪功能

说明

pg_untrace ( resource $connection = ? ) : bool

停止由 pg_trace() 启动的追踪功能。connection 指定了被追踪的连接号，默认为上一个打开的连接。

本函数总是返回 true。

参见 pg_trace()。
