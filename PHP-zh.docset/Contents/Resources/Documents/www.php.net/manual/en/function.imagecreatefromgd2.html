imagecreatefromgd2

(PHP 4 >= 4.0.7, PHP 5, PHP 7, PHP 8)

imagecreatefromgd2 — 从 GD2 文件或 URL 新建一图像

说明

imagecreatefromgd2(string $filename): resource

从 GD2 文件或 URL 新建一图像。

Tip

如已启用fopen 包装器，在此函数中， URL 可作为文件名。关于如何指定文件名详见 fopen()。各种 wapper 的不同功能请参见 支持的协议和封装协议，注意其用法及其可提供的预定义变量。

参数

filename

GD2 图像的路径。

返回值

成功后返回图象对象,失败后返回 false。

范例

Example #1 imagecreatefromgd2() 例子

<?php
// 加载 gd2 图像
$im = imagecreatefromgd2('./test.gd2');


// 在图像上应用效果。
// 在这个例子中，如果版本为 PHP5+ 则反转图像颜色
if(function_exists('imagefilter'))
{
    imagefilter($im, IMG_FILTER_NEGATE);
}

// 保存图像
imagegd2($im, './test_updated.gd2');
imagedestroy($im);
?>
