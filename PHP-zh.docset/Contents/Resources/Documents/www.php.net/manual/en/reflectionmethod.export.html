ReflectionMethod::export

(PHP 5, PHP 7)

ReflectionMethod::export — 输出一个回调方法

说明

public static ReflectionMethod::export ( string $class , string $name , bool $return = false ) : string

Exports a ReflectionMethod.

Warning

本函数还未编写文档，仅有参数列表。

参数

class

类名称

name

方法名称

return

设为 true 时返回导出结果，设为 false（默认值）则忽略返回。

返回值

如果参数 return 设为 true，导出结果将作为 string 返回，否则返回 null。
