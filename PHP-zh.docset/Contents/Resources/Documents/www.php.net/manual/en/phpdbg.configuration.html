运行时配置

这些函数的行为受 php.ini 中的设置影响。

phpdbg 配置选项
名字 默认 可修改范围 更新日志
phpdbg.eol 2 PHP_INI_ALL  
phpdbg.path   6  

这是配置指令的简短说明。

phpdbg.eol mixed

The kind of line ending to use for output. For setting the value, one of the string aliases must be used.

int Value string Alias
0 CRLF, crlf, DOS, dos
1 LF, lf, UNIX, unix
2 CR, cr, MAC, mac

phpdbg.path string
