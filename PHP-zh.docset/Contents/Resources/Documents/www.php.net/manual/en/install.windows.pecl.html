PECL

PECL 扩展在 Windows 版本中是预编译的，可从以下网站获得 » http://windows.php.net/downloads/pecl/releases/

某些扩展使用了某些 Unix 系统所特有的功能，因此在 Windows 上不可用。除此之外，所有的扩展都可以在 Windows 上使用。
