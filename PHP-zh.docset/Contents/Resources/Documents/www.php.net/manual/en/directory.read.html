Directory::read

(PHP 4, PHP 5, PHP 7)

Directory::read — 从目录句柄中读取条目

说明

public Directory::read ([ resource $dir_handle ] ) : string

与 readdir() 函数功能一样, 只是 dir_handle 默认为 $this 变量。
