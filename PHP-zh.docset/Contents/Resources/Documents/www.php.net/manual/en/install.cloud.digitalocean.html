DigitalOcean

DigitalOcean 提供以下平台来安装 PHP， 并在其云托管基础架构上运行 Web 应用程序。

  • » Cloudways： 一键部署主要的 PHP 应用程序： WordPress、Magento、Drupal、Laravel 等等。

  • » Droplet： 使用虚拟机在 »  Linux 服务器上安装 LAMP 堆栈 。

  • » App Platform： 使用托管基础架构快速构建、部署和扩展应用程序。 了解 »  如何在应用平台上运行 PHP 应用程序 。

  • » Functions： 无服务器平台允许开发人员在不预配或管理服务器的情况下运行代码。 PHP 被原生支持。 了解 »  如何使用 PHP 创建无服务器函数 。
