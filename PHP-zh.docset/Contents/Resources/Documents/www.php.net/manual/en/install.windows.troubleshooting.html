Windows 下 PHP 故障排除

检查临时目录权限

  1. 在"文件管理器"中右键单击"临时目录"以获得权限。

  2. IIS, 检查用户的 IIS_User 是否有修改文件的权限。可以从配置或 php info 中获取临时目录的位置。
