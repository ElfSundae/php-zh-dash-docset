fann_create_sparse

(PECL fann >= 1.0.0)

fann_create_sparse — 创建一个标准的反向传播神经网络，该网络不是全连接。

说明

fann_create_sparse ( float $connection_rate , int $num_layers , int $num_neurons1 , int $num_neurons2 [, int $... ] ) : ReturnType

创建一个标准的反向传播神经网络，该网络不是全连接。

参数

connection_rate

连接率控制着在网络中将会有多少连接，如果连接率设置为1，那么这个网络就是全连接网络，但是如果设置为 0.5 将会设置一半的连接。连接率为1的结果和使用 fann_create_standard()函数的效果是一样的。

num_layers

神经网络层数，包括输入输出层。

num_neurons1

第一层网络的神经数。

num_neurons2

第二层网络的神经数。

...

其它层网络的神经数。

返回值

返回一个神经网络资源，错误则返回 FALSE .

参见

  • fann_create_sparse_array() - 创建一个标准的反向传播神经网络，该网络使用一个表示每层大小的数组来构造，但是并不是全连接的。
  • fann_create_standard() - 创建标准的全连接反向传播神经网络。
  • fann_create_shortcut() - 创建一个含快捷连接而非全连接的标准反向传播神经网络。
