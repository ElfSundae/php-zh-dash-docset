ReflectionEnumBackedCase 类

(PHP 8 >= 8.1.0)

简介

ReflectionEnumBackedCase 类报告了一个回退枚举条目（Enum backed case）的信息， 具有对应的标量值。

public $name ;
public $class ;
/* 方法 */
public getBackingValue ( ) : string|int
/* 继承的方法 */

public ReflectionEnumUnitCase::getValue ( ) : UnitEnum
}
