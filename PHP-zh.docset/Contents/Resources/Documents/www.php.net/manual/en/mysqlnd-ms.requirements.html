需求

PHP 5.3.6 或更高版本，一些建议使用的功能，需要 PHP 5.4.0 或更高版本。

mysqlnd_ms 主从同步和负载均衡插件，可以支持所有的 PHP 应用程序，并且支持所有的 PHP MySQL 扩展 (mysqli, mysql, PDO_MYSQL)。 所有的 PHP MySQL 扩展必须配置为使用 mysqlnd 并且在 mysqlnd 中使用 mysqlnd_ms。
