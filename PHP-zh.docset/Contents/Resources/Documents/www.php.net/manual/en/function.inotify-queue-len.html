inotify_queue_len

(PECL inotify >= 0.1.2)

inotify_queue_len — 如果有待处理事件，返回大于零的数字

说明

inotify_queue_len(resource $inotify_instance): int

可以通过此函数知道 inotify_read() 是否会阻塞。如果返回大于零的数字，表示有待处理的事件，inotify_read() 不会阻塞。

参数

inotify_instance

inotify_init()返回的资源

返回值

如果有待处理事件，返回大于零的数字。
