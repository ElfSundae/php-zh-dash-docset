finfo_set_flags

finfo::set_flags

(PHP >= 5.3.0, PHP 7, PHP 8, PECL fileinfo >= 0.1.0)

finfo_set_flags -- finfo::set_flags — 设置 libmagic 配置选项

说明

过程化风格

finfo_set_flags(finfo $finfo, int $flags): true

面向对象风格

public finfo::set_flags(int $flags): bool

此函数用来设置 Fileinfo 选项。 这些选项也可以在调用 finfo_open() 或者其他 Fileinfo 函数时直接指定。

参数

finfo

经 finfo_open() 返回的 finfo 实例。

flags

一个 Fileinfo 常量 或多个 Fileinfo 常量 进行逻辑或运算。

返回值

总是返回 true。

更新日志

版本 说明
