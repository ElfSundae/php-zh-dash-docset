Thread::detach

(PECL pthreads < 3.0.0)

Thread::detach — 执行

Warning

pthreads v3 中已经移除此方法.

说明

public Thread::detach ( ) : void

从调用上下文中将引用线程分离出来，非常危险！

Warning

本方法会引发未定义的、不安全的行为。 通常情况下不会用到本方法，提供这个方法主要是出于完备性以及高级用法的考虑。

参数

此函数没有参数。
