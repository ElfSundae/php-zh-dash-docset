rrd_tune

(PECL rrd >= 0.9.0)

rrd_tune — 调整 RRD 数据库文件头选项

说明

rrd_tune(string $filename, array $options): bool

更改 RRD 数据库头文件中的一些选项，例如：重命名数据源等。

参数

filename

RRD 数据库文件名。

options

要修改的 RRD 数据库文件属性选项。请参阅 rrd tune 手册页获取详细信息。

返回值

成功时返回 true， 或者在失败时返回 false。
