imagecolorstotal

(PHP 4, PHP 5, PHP 7, PHP 8)

imagecolorstotal — 取得一幅图像的调色板中颜色的数目

说明

imagecolorstotal ( resource $image ) : int

本函数返回指定图像的调色板中的颜色数目。

参见 imagecolorat() 和 imagecolorsforindex()。
